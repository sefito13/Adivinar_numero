# Instrucciones
- Tienes diferentes modos de juego(Nivel de dificultad).
- Tu ingresas el numero hasta adivinarlo.
- Dependiendo el nivel de dificultad tienes:

    - 5 Turnos en Dificil.
- Tienes pistas donde te dira si es menor o mayor el numero a adivinar.
